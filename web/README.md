# Arcanum Web

Initial `M0.4` and `M0.5` scaffold for the Aurora web character builder.

## Included

- Next.js App Router structure with TypeScript
- Placeholder routes for the approved initial IA
- Shared app shell and styling
- Supabase browser/server helper modules
- `.env.example` for local environment setup

## Next Steps

1. Install dependencies.
2. Copy `.env.example` to `.env.local` and provide Supabase credentials.
3. Start the app with `npm run dev` or `pnpm dev`.
